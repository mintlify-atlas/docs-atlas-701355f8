## Welcome to Flask-Discord

Flask-Discord is a Discord OAuth2 extension for Flask that simplifies integrating Discord authentication into your Flask applications. It provides an easy-to-use interface for implementing Discord's OAuth2 flow, managing user sessions, and accessing Discord API resources.

## Key Features

- **Easy OAuth2 Integration**: Simplified Discord OAuth2 flow with Flask session management
- **User Data Access**: Fetch Discord user profiles, connections, and guilds
- **Bot Integration**: Support for bot scope with customizable permissions
- **Flexible Caching**: Built-in user caching with configurable cache backends
- **Decorator Support**: Protect routes with `@requires_authorization` decorator
- **Error Handling**: Custom exception classes for common OAuth2 errors

## Requirements

Flask-Discord requires the following dependencies:

- Flask
- requests_oauthlib
- cachetools
- pyjwt>=2.4.0
- requests
- oauthlib

<Note>
  All dependencies are automatically installed when you install Flask-Discord via pip.
</Note>
